# Compliance Document Templates

This directory holds non-legal stub templates for common enterprise
procurement paperwork. They are starting points for discussion with
your own counsel, not signed legal documents.

| File             | Purpose                                                |
|------------------|--------------------------------------------------------|
| `DPA.md`         | Data Processing Addendum outline (EU/UK GDPR).         |
| `MSA.md`         | Master Services Agreement outline.                     |
| `SOC2.md`        | SOC 2 Type II readiness / control mapping notes.       |
| `hipaa-baa.md`   | HIPAA Business Associate Agreement outline.            |

uxinspect uses a controls platform (Vanta or Drata) to evidence the
operational side of these documents. The text here is intentionally
thin — it tells a buyer what to expect, points them at the control
platform for audit artefacts, and flags where legal review is
required before signature.

> **Do not sign any of these as-is.** Every template in this folder
> contains `TBD` placeholders. Run the draft through your attorney,
> fill the blanks, and capture the signed PDF in your CRM.
